# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름

```
시작
반복 (사용자가 게임을 그만둘 때까지):
    게임 시작 함수 호출
    "다시 하시겠습니까? (1: 예, 0: 아니오)" 출력
    사용자 입력 받기 (play_again)
    만약 play_again이 0이면:
        반복 종료
반복 끝
"게임을 종료합니다!" 출력
종료
```

## 2. 핵심 기능 1: 게임 시작 함수 (start_game)

```
시작
    시도 횟수 = 0
    정답 = 1부터 100 사이의 랜덤 숫자 생성
    
    "=== 숫자 맞추기 게임 ===" 출력
    "1부터 100 사이의 숫자를 맞춰보세요!" 출력
    
    반복 (정답을 맞출 때까지):
        시도 횟수 증가
        "숫자를 입력하세요: " 출력
        사용자 입력 받기 (guess)
        
        만약 guess가 정답보다 작으면:
            "UP! 더 큰 숫자입니다." 출력
        만약 guess가 정답보다 크면:
            "DOWN! 더 작은 숫자입니다." 출력
        만약 guess가 정답과 같으면:
            "정답입니다!" 출력
            "총 시도 횟수: [시도 횟수]회" 출력
            반복 종료
    반복 끝
종료
```

## 3. 핵심 기능 2: 랜덤 숫자 생성 함수 (generate_random)

```
시작
    현재 시간을 기준으로 난수 초기화 (srand)
    1부터 100 사이의 랜덤 숫자 생성 (rand)
    숫자 반환
종료
```

## 4. 최적화 고려사항
- 입력 검증: 1~100 범위를 벗어난 숫자 입력 시 경고 메시지
- 잘못된 입력 처리: 문자 입력 시 오류 처리
