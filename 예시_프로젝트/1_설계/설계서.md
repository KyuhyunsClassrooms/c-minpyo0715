# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
**숫자 맞추기 게임**
- 컴퓨터가 1~100 사이의 숫자를 하나 정합니다.
- 사용자는 숫자를 맞출 때까지 계속 입력합니다.
- 힌트(UP/DOWN)를 제공하여 맞추기 쉽게 합니다.

## 2. 핵심 기능 목록
1. 컴퓨터가 랜덤 숫자 생성 (1~100)
2. 사용자로부터 숫자 입력받기
3. 입력한 숫자와 정답 비교
4. 힌트 제공 (UP: 더 큰 숫자, DOWN: 더 작은 숫자)
5. 정답을 맞추면 시도 횟수 출력
6. 게임 반복 (다시 하기 기능)

## 3. 데이터 구조 설계
**방법: 배열 사용 (기본)**
```c
int answer;        // 컴퓨터가 생성한 정답 숫자
int guess;         // 사용자가 입력한 숫자
int tries;         // 시도 횟수
int play_again;    // 게임 반복 여부 (1: 예, 0: 아니오)
```

**추가 설명:**
- 이 프로젝트는 간단하여 배열이 꼭 필요하지 않습니다.
- 일반 변수만으로 충분히 구현 가능합니다.
